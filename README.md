## 📖 Project Overview

The Ultimate Biryani Handbook is an interactive culinary encyclopedia built using React.js. It provides detailed information about various types of biryanis, their origins, ingredients, preparation techniques, nutritional benefits, and regional variations.

The application covers biryanis from all 28 Indian states, South Indian specialties, and international biryani varieties from multiple countries, making it one of the most comprehensive biryani information platforms.

## 🚀 Installation

### Clone Repository

git clone https://github.com/charanepuri/ultimate-biryani-handbook.git

### Navigate to Project

cd ultimate-biryani-handbook

### Install Dependencies

npm install

### Run Development Server

npm run dev

### Build for Production

npm run build
